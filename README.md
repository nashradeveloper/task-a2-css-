# 🎨 Html CSS Layout Project

A minimal, typographic web page built with pure HTML & CSS.

---

## ⚡ Tech
- HTML5 & CSS3
- No frameworks, no JavaScript

## 🧠 CSS Concepts Used

| Property | Purpose |
|---|---|
| `display: flex` | Header & section layout |
| `justify-content: space-between` | Space elements apart |
| `margin: auto` | Center blocks horizontally |
| `text-transform: uppercase` | All-caps styling |
| `text-decoration: underline` | Underlined headings |
| `object-fit: cover` | Clean image cropping |
| `border-radius` | Rounded image corners |
| `font-size: em` | Scalable large typography |
| `text-align: end` | Right-aligned text |
| `box-sizing: border-box` | Consistent sizing |

## 🎨 Color Palette
- Primary — `#6d0e11` (deep red)
- Muted — `#AB9696` (dusty rose)

## 🚀 Run
Just open `index.html` in a browser.
